# Get Your credentials.json File

## Quick Steps (5 minutes)

1. **Go to Google Cloud Console**
   https://console.cloud.google.com/

2. **Create/Select Project**
   - Click "Email Summarizer" or create new project
   - Make sure Gmail API is enabled

3. **Get OAuth Credentials**
   - Go to: APIs & Services → Credentials
   - Click "Create Credentials" → OAuth 2.0 (Desktop app)
   - Click the **Download** button (⬇️ icon)
   - Save the downloaded file

4. **Rename & Place**
   - Rename the file to: `credentials.json`
   - Move it to this folder

5. **Done!**
   Run: `python email_customtkinter_gui.py`

---

If you already downloaded it before, check your Downloads folder for a file like:
- `client_secret_*.json`
- Or any JSON file from Google Cloud

**Just rename it to `credentials.json` and put it here!**
